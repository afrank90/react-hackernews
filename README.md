## Description
This is learning react codebase! It's a simple UI to fetch data from the API.

### Stack
- ES6
- React v16
- Prop Types
- Jest
- Enzyme
- Axio

### Installation
- Clone repository
- Navigate to the app root folder
- Run node installation command
```sh
npm install
```
- Start the app
```sh
npm start
```

### Tests
```sh
npm test
```

### ESLint
```sh
npm run lint src/components
```
